Basic Chat Bot 🤖

A simple Basic Chat Bot built using Python that interacts with users through predefined responses.
The chatbot responds to user inputs like greetings, questions, and simple conversations using conditional statements and loops.

📌 Features
Interactive text-based chatbot
Responds to basic user queries
Predefined conversational replies
Beginner-friendly Python project
Simple and easy-to-understand code
🛠️ Technologies Used
Python 3
📂 Project Structure
basic-chatbot/
│── chatbot.py
│── README.md
▶️ How to Run the Project
Install Python on your system.
Download or clone this repository.
Open terminal or command prompt.
Run the following command:
python chatbot.py
🎯 Functionalities
Greets the user
Answers simple questions
Responds to user inputs
Continues conversation until exit command
Provides an interactive chatting experience
