Project Statement – Hangman Game
Problem Statement

Traditional word guessing games help improve vocabulary, logical thinking, and problem-solving skills. The objective of this project is to develop a simple and interactive Hangman Game using Python that allows users to guess a hidden word within a limited number of attempts.

The game randomly selects a word from a predefined list and displays blank spaces representing each letter. The player guesses one letter at a time. Correct guesses reveal the matching letters in the word, while incorrect guesses reduce the available attempts. The game continues until the player either successfully guesses the word or runs out of chances.

This project is designed to strengthen programming fundamentals such as loops, conditional statements, string manipulation, lists, functions, and user input handling in Python. It also provides beginners with practical experience in building a command-line based mini project.

Objectives
To build an interactive word guessing game using Python.
To implement game logic using loops and conditions.
To enhance problem-solving and logical thinking skills.
To understand the use of randomization and string operations.
To create a beginner-friendly mini project for learning Python development.
Technologies Used
Python 3
Random Module
Command Line Interface (CLI)
Expected Outcome

The final application will allow users to:

Play a fully functional Hangman Game.
Guess letters interactively.
Receive instant feedback on guesses.
Win or lose based on the number of attempts remaining.

The project demonstrates the practical implementation of core Python concepts through a fun and engaging game.
