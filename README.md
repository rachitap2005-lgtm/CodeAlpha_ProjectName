# 🤖 Basic Chatbot using Python

## 📌 Project Overview
This project is a simple rule-based chatbot developed using Python. The chatbot interacts with users by responding to predefined questions and basic conversational inputs. It is designed to simulate human-like conversation for common queries.

## 🎯 Objective
The objective of this project is to understand the fundamentals of chatbot development, user interaction handling, and conditional logic implementation using Python.

## ✨ Features
- Interactive text-based conversation
- Responds to common greetings and questions
- Handles predefined user inputs
- Simple and beginner-friendly Python code
- Easy to customize with new responses

## 🛠 Technologies Used
- Python
- Conditional Statements (`if-elif-else`)
- Functions
- Loops
- String Handling
