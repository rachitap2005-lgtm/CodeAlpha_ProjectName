📘 README File
Python Mini Projects Collection

This repository contains three beginner-friendly Python projects:

Hangman Game
Stock Portfolio Tracker
Basic Chatbot

These projects are useful for learning Python fundamentals such as loops, conditions, functions, lists, dictionaries, file handling, and user interaction.

📂 Projects Included
1️⃣ Hangman Game
📌 Description

The Hangman Game is a word-guessing game where the player tries to guess a hidden word letter by letter before running out of attempts.

✨ Features
Random word selection
User-friendly gameplay
Tracks incorrect guesses
Displays remaining attempts
Beginner-friendly logic
🛠 Technologies Used
Python
Lists
Loops
Conditional Statements
Random Module
▶️ How to Run
python hangman.py
📷 Example Output
Enter a letter: a
Correct Guess!

Word: _ a _ _ _

Attempts Left: 5
2️⃣ Stock Portfolio Tracker
📌 Description

✨ Features

Calculate total investment value
Store portfolio data
File handling support
Simple and efficient code
🛠 Technologies Used
Python
Dictionary
File Handling
Basic Arithmetic Operations
▶️ How to Run
