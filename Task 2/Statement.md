Project Statement – Stock Portfolio Tracker
Problem Statement

Managing stock investments manually can become difficult when handling multiple stocks, quantities, and prices. Investors often need a simple system to store stock information, calculate total investment values, and maintain records efficiently.

The objective of this project is to develop a Stock Portfolio Tracker using Python that helps users manage and track their stock investments in a simple and organized way. The application allows users to enter stock details such as stock name, quantity, and price, then automatically calculates the total investment value using basic arithmetic operations.

The project utilizes Dictionary Data Structure for storing stock information and File Handling for saving portfolio records permanently into a text file. This project is designed for beginners to understand practical implementation of Python concepts through a finance-related application.

Objectives
To develop a simple stock portfolio tracking system using Python.
To implement dictionary data structures for storing stock details.
To perform investment calculations using arithmetic operations.
To use file handling for permanent data storage.
To improve programming and logical thinking skills through a real-world project.
Technologies Used
Python 3
Dictionary
File Handling
Command Line Interface (CLI)
Expected Outcome

The final application will allow users to:

Add stock details easily
Store stock quantity and prices
Calculate total investment values
Save portfolio information into a file
View organized stock records

The project demonstrates practical implementation of Python fundamentals including dictionaries, arithmetic calculations, and file handling in a real-world financial application.
